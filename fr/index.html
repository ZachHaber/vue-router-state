Vue Router

Routeur officiel pour Vue.Js

Get Started →

Vue Router est le router officiel pour Vue.js (opens new window). Il s'intègre aisément avec Vue.js pour faire des applications mono page avec Vue.js. Fonctionnalités incluses:

  • Vues et routes imbriquées
  • Modulaire, configuration basée sur les composants
  • Paramètres de route, de requête
  • Effets de transition de vues basées sur le systeme de transition de Vue.js
  • Gestion fine de la navigation
  • Classes CSS pour liens actifs
  • Mode HTML5 de la gestion de l'historique ou mode par hash, avec solution de repli automatique pour IE9
  • Gestion du scroll

Pour commencer ou jouer avec les exemples (opens new window) (voir README.md (opens new window) pour les faire fonctionner).
